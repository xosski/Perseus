# Perseus Knowledge Folder

Drop local reference material here when you want Perseus to ingest it into the knowledge store.

Supported file types: `.txt`, `.md`, `.docx`, `.py`, `.json`, `.yaml`, `.yml`, `.csv`, `.html`, `.htm`, `.log`, `.typed`, and no-extension text files.

Use the desktop app's `Knowledge Ingest` tab and click `Ingest Folder`, or call `PortableLLM.ingest_folder("knowledge")` from Python.
